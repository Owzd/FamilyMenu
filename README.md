# FamilyMenu
自己设计并实现的菜谱查询功能App，可以称之为家庭菜谱。

## 背景
之前做的App一直使用udp/tcp的通讯方式，但http协议依然很普遍且更合理，此项目包含http通讯及解析复杂格式json数据。

## 功能描述
* App引导
* 按菜名查询
* 按标签查询
* 设置个人信息

## 技术栈
* 原生安卓
* http
* 聚合数据
